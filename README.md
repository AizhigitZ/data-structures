# DSA Dashboard — Quick Start

This repository contains a React dashboard component `DSA_Dashboard.jsx` and a minimal Vite setup to run and build it.

Requirements:
- Node.js (16+ recommended)

Install & run locally:

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```

Notes:
- `index.js` mounts the default export from `DSA_Dashboard.jsx`.
- This setup uses Vite for fast development and simple builds.
